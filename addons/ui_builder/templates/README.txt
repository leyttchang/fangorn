Drop .tscn scene files in this folder to make them appear in the Quick
Layout dock's "Template" dropdown (UI Builder top bar).

To add your own: build something in the UI Builder canvas (or anywhere in
the Scene tree), select the root Control node of what you want to reuse,
and click "Save Selected as Template..." in the UI Builder top bar. It
saves here by default.

Click "Refresh" after adding new files by hand.
