Drop Theme resources (.tres) in this folder to make them appear in the
Quick Layout dock's "Theme Preset" dropdown.

To create one: in Godot, right-click in the FileSystem dock inside this
folder -> New Resource -> Theme, then edit it in the Theme editor and save
it here (e.g. "dark_ui.tres", "rounded_light.tres").

Click "Refresh" in the dock after adding new files.
